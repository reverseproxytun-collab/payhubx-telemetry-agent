# PayHubX Cloud Agent
Official lightweight telemetry agent for PayHubX Infrastructure Monitoring.

## Installation
`wget https://raw.githubusercontent.com/payhubx-telemetry-agent/master/install.sh`
`chmod +x install.sh && ./install.sh`

## Configuration
Edit `/etc/payhubx/agent.yaml` and paste your License Token from the PayHubX dashboard.
